Tupperbox Guide

What is Tupperbox?

Tupperbox is a Discord bot that allows you to simulate having multiple accounts, with unique names and avatars, without having to log in and out every time.

How does Tupperbox work?

Tupperbox watches the server, and when it sees the brackets that tell it that a tupper is talking, it takes the message text, deletes your message, and has the tupper re-send the message with the proper name and avatar.

What is a tupper?

A tupper is basically an alternate account or user. It can be for your alter, your fursona, your RP character, or anything else you’d like to have an alternate identity for.

How do I set up Tupperbox?

All you need to do to get started is create a new tupper. You can create a new tupper by using the tul!register command, followed by the name of the tupper, and the brackets to tell Tupperbox that the tupper is talking. Names that contain more than one word should be placed in single or double quotes. The brackets can be anything, and on either side of the text. Just type the word text with the brackets set up around it. You can also attach an image to this message, and it will be the avatar for that tupper. Once you set up your tupper, start sending messages with your brackets, and Tupperbox will handle the rest!

Examples:

tul!register “Steven Smith” >text< will create a tupper where this text >Hello, world!< will appear like this:

tul!register Jessica text$$ will create a tupper where this text Hello, world!$$ will appear like this:

tul!register Mario ::text:: with this image attached will create a tupper where this text ::It’s-a me!:: will appear like this:

How do I delete a tupper?

Use the command tul!remove followed by the name of the tupper you wish to delete, and tupper will unregister the tupper.

How do I rename a tupper?

Use the command tul!rename followed by the tupper’s old name, then the tupper’s new name.

How do I change a tupper’s avatar?

Use the command tul!avatar followed by the tupper’s name with the new avatar attached.

How do I see all my tuppers?

The command tul!list will print out a list of your tuppers and some information about them. tul!listng will do the same thing, but without group info.

Can I have Tupperbox change all my messages?

Yes! just use the command tul!auto followed by the name of the tupper you’d like to be active with. You can change the active tupper with the same message, just change the name. To turn it off, use tul|auto off.

Why can’t I edit my tupper’s messages? /Why can’t I give my tupper roles and colors?

Because tuppers aren’t actually unique users or bots, they’re webhooks. Unfortunately, this is a Discord thing, and not something we can change.

Is there a limit to how many tuppers I can have?

Technically? Yes. But it’s almost impossible to hit unless you’re abusing the bot.

My question isn’t answered here/I need help with something more specific

Try tul!help followed by the name of the command (so, tul!help register for help with the register command. If you can’t find your help there, you can always join the Tupperbox support server.
